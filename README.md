# Закрывающий тег — сайт об изучении верстки

«Закрывающий тег» — это итоговый учебный проект, завершающий модуль «Верстка» в рамках курса «Фронтенд-разработчик» от «Яндекс Практикума». Представляет из себя страницу с резиновым интерфейсом и постами, которые можно лайкать.

## Задачи

В рамках этого проекта я выполнил следующие задачи.

- Настроил проект, подготовил SVG-графику и добавил фавиконки.

- Подключил и настроил вариативный шрифт.

- Создал фоновый паттерн на основе сложного множественного градиента.

- Реализовал «резиновую» верстку с помощью функции `clamp()`.

- Добавил лейблы поверх ихображений с помощью абсолютного позиционирования, стилизовал их при помощи свойства `text-stroke` через директиву `@supports` и оставил `text-shadow` в качестве фоллбэка.

- Реализовал механику открытия модального окна с помощью элемента `dialog` и стилизовал фон через псевдоэлемент `backdrop`.

- Реализовал адаптивность интерфейса с помощью медиазапросов.

- Стилизовал изображения с помощью CSS-фильтров.

- Анимировал элементы интерфейса в различных состояниях с помощью `transition`.

- Анимировал SVG-графику через измение свойств различных путей (path) посредством `transition` и `@keyframes`.
